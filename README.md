Search the text of the documents released from the Jeffery Epstein investigation that were released by the House Oversight Committee on Nov. 12, 2025. 

Two run locally put the two python scripts and the sqlite database in the same directory and run the web ui script and open browser to localhost:5000

You can get those files here: https://oversight.house.gov/release/oversight-committee-releases-additional-epstein-estate-documents/

If you want to see the original image scans, download the 12 image zips from the House Oversight Commitee and unzip to project folder, then click on "see original image."
